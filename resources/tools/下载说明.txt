===========================================
IconsExtract 下载和安装说明
===========================================

📥 下载地址：

官方网站：
https://www.nirsoft.net/utils/iconsext.html

直接下载：
https://www.nirsoft.net/utils/iconsext.zip

===========================================

📂 安装步骤：

1. 下载 iconsext.zip 压缩包
2. 解压文件
3. 找到 iconsext.exe
4. 复制到当前文件夹（resources\tools\）

最终路径应该是：
D:\projects\fingertips-ai\resources\tools\iconsext.exe

===========================================

✅ 验证安装：

重新运行应用后，日志中应该显示：
"✓ IconsExtract found"

如果显示：
"✗ IconsExtract not found"
说明文件位置不对。

===========================================

💡 特点：

✅ 从 .exe 和 .dll 提取高质量图标
✅ 自动提取多种尺寸的图标
❌ 不支持 .ico 文件（会自动跳过）
❌ 不支持 .lnk 文件（会自动跳过）

===========================================

📝 注意事项：

- IconsExtract 专门用于 PE 文件（可执行文件和 DLL）
- 它不能处理纯 .ico 图标文件或 .lnk 快捷方式
- 系统已自动过滤，不会产生错误

如果看到日志：
"ℹ Skipping IconsExtract for .ico file"
这是**正常行为**，不是错误！

系统会自动使用其他方法（Electron API 或 PowerShell）处理这些文件。

===========================================

🆘 如果遇到问题：

1. 确认文件名正确：iconsext.exe
2. 确认路径正确：在 resources\tools\ 目录下
3. 查看日志中的完整路径
4. 确保文件没有被杀毒软件拦截
5. 尝试重新下载

===========================================

📊 提取流程：

1. 首先尝试 Electron API（快速，适合大多数情况）
2. 如果质量不足（< 200分），尝试 IconsExtract
3. 如果 IconsExtract 失败，使用 PowerShell Enhanced
4. 最后使用 PowerShell Simple 作为兜底
5. 自动选择质量最高的图标
