# 常用语速记 (Quick Phrases)

> 快速复制常用语到剪切板，双击即复制，支持分类管理和搜索

## ✨ 功能特性

- **双击复制** - 双击常用语卡片即可快速复制到剪切板
- **分类管理** - 支持工作、生活、邮件、社交等多种分类
- **实时搜索** - 快速搜索定位需要的常用语
- **分类过滤** - 按分类筛选，快速找到目标内容
- **编辑删除** - 随时编辑或删除已有常用语
- **使用统计** - 记录使用次数和最后使用时间

## 🎨 界面设计

采用 2025 年最新设计美学：

- **Glassmorphism** - 毛玻璃效果，层次分明
- **Bento Grid** - 卡片式布局，清晰直观
- **流畅动画** - 丝滑的交互动画效果
- **分类色彩** - 不同分类用不同颜色标识

## 📋 使用方法

1. **打开插件** - 从插件列表点击执行
2. **添加常用语** - 点击右上角「添加常用语」按钮
3. **复制使用** - 双击任意常用语卡片即可复制
4. **搜索过滤** - 使用搜索框或分类标签快速定位
5. **编辑删除** - 悬停卡片后点击编辑或删除按钮

## 🏷️ 默认分类

| 分类 | 颜色 | 适用场景 |
|------|------|----------|
| 工作 | 紫色 | 工作相关的常用语 |
| 生活 | 绿色 | 日常生活用语 |
| 邮件 | 橙色 | 邮件模板、签名等 |
| 社交 | 粉色 | 社交媒体、聊天用语 |
| 其他 | 灰色 | 其他未分类内容 |

## ⚙️ 权限说明

| 权限 | 用途 |
|------|------|
| `window` | 创建常用语管理窗口 |
| `notification` | 复制成功时显示通知 |
| `clipboard` | 将常用语写入剪切板 |

## 💡 使用技巧

1. **快速复制** - 双击卡片是最快的复制方式
2. **搜索技巧** - 可以搜索内容或分类名称
3. **分类选择** - 添加时选择合适的分类便于后续管理
4. **多行内容** - 支持添加多行的常用语内容
